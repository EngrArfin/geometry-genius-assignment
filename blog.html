    4.What is callBack Function? What works with the call Back function

  • Answer to the Q NO:-2

    •             JavaScript callback is a function which is to be executed after
             another function has finished execution.
            
            Call Back function is a function passed into another function as an argument,
             which is then invoked inside the outer function to complete some kind of routine or action.
            
            Callbacks allow you to continue executing code while the operation is being
             executed in the background.
